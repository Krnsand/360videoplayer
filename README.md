# 360째 Video Player

This project is a simple 360째 video player built with JavaScript and Video.js (with the videojs-vr plugin).

## Getting Started

1. Place your 360째 video file (e.g., `sample360.mp4`) in the project directory.
2. Open `index.html` in your browser, or run a local server (`npm start` if you have Node.js installed).

## Features
- 360째 video playback
- Mouse and touch controls for looking around

## Dependencies
- [Video.js](https://videojs.com/)
- [videojs-vr plugin](https://github.com/videojs/videojs-vr)

## How to Use
- Replace the sample video source in `index.html` with your own 360 video file.
